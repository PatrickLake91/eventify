About Eventify

A prototype web application for managing public events.

Website description

Eventify is designed for a single event organiser to manage event information. The organiser can add event details and view a list of events that have already been created.

This prototype focuses on the core front-end functionality required for the assessment.

Developer

This prototype was developed for the DAT6006 Full Stack Development assessment.

It is built using HTML, Bootstrap, CSS and JavaScript, with browser localStorage used to store event data.

Design and architecture approach

The interface follows a consistent layout across all pages, with clear navigation and form validation. The conceptual design follows a simple MVC-style structure, where HTML pages act as views and JavaScript handles user interaction and data handling.
