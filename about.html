About Eventify

A prototype web platform for advertising public events.

Website description

Eventify is designed as a simple event advertising platform for a single event organiser. The prototype allows the organiser to add event information and view events that have previously been added.

The implemented version focuses on the required front-end prototype activities. The wider full-stack design is represented through architecture documentation, including an ERD and sequence diagrams.

Developer

This prototype was developed for the DAT6006 Full Stack Development PRES assessment.

The design applies Bootstrap, custom CSS, JavaScript interactivity, local browser storage, and usability principles.

Design and architecture approach

The interface uses a consistent layout, clear navigation, form validation, responsive design and visual feedback. The conceptual architecture follows an MVC-inspired structure where pages act as views, JavaScript handles interaction logic, and localStorage simulates prototype data persistence.
